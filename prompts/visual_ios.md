# Visual Verification Block — iOS

This block is injected for tasks with the `ios` label.

## Additional Review Requirements

### UI Guidelines
- Follows Apple Human Interface Guidelines?
- Uses system fonts and standard controls where appropriate?
- Supports Dynamic Type for text scaling?
- Handles safe area insets correctly?

### Accessibility
- VoiceOver labels on all interactive elements?
- Sufficient color contrast?
- Respects system accessibility settings (bold text, reduce motion)?

### Device Compatibility
- Works on iPhone SE (small screen) through iPhone Pro Max?
- Handles orientation changes if applicable?
- Supports dark mode?
